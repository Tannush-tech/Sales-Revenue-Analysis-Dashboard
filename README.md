# Sales Revenue Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes sales and revenue data to provide meaningful business insights. It helps track sales performance, profit, quantity, and regional trends.

## Tools Used
- Power BI
- Excel

## Dataset
- Sample Superstore Sales Dataset

## Dashboard Features
- Total Sales
- Total Profit
- Total Quantity
- Sales by Region
- Sales by Category
- Sales Trend
- Top Selling Products
- Region Filter

## Project Outcome
- Sales performance analysis
- Revenue insights
- Regional comparison
- Business decision support
